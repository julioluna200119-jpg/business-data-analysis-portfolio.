# Portfolio de Análisis de Datos de Negocio
Análisis estratégicos enfocados en FP&A y Operaciones.

## 📊 Proyectos Destacados

### 1. Análisis Financiero (Adventure Works)
* **Objetivo:** Analizar rentabilidad y ROI por país.
* **Resultado Clave:** Identifiqué a **USA** como el país con mayor ROI (**75.7%**), recomendando reasignar presupuesto a esta región.
* **Tecnología:** SQL (Joins, Agregaciones).

### 2. Embudo de Conversión y Retención (Mercado Libre)
* **Objetivo:** Optimizar el flujo de usuarios y medir lealtad.
* **Resultado Clave:** Detecté una caída crítica del **89%** al agregar productos al carrito. Propuse incentivos en el Día 21 para frenar la pérdida de usuarios en el Día 28.
* **Tecnología:** SQL (CTEs, Análisis de Cohortes).

### 3. Tendencias de Economía y Tráfico
* **Objetivo:** Relacionar indicadores macroeconómicos con el tráfico de la plataforma.
* **Tecnología:** Python (Pandas, Visualización).
